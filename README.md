# 🚀 Asteroids Unity Game

A classic Asteroids-style arcade game built in Unity. Pilot your spaceship, destroy asteroids, and survive as long as possible while racking up points.
---
## 🎮 Features

- Classic arcade-style asteroid gameplay
- Smooth spaceship movement and rotation
- Shooting mechanics with projectiles
- Asteroid spawning and splitting system
- Score system based on destroyed asteroids
- Increasing difficulty over time (optional)
- Simple 2D space visuals
---
## 🕹️ Controls
- Move Forward: W / Up Arrow
- Rotate Left: A / Left Arrow
- Rotate Right: D / Right Arrow
- Shoot: Spacebar
---
## 💥 Gameplay
1. Control your spaceship in open space
2. Avoid crashing into asteroids
3. Shoot asteroids to destroy them
4. Large asteroids split into smaller ones
5. Survive as long as possible and get the highest score
---
## 🧩 Game Mechanics
- **Asteroids**
  - Spawn randomly around the map
  - Split into smaller pieces when shot
  - Destroy player on collision
- **Player Ship**
  - Physics-based movement
  - Rotation + thrust control
  - Can shoot projectiles
- **Bullets**
  - Travel in straight direction
  - Destroy asteroids on impact
---
## 🛠️ Built With

- Unity Engine (2D)
- C# scripting
- Rigidbody2D physics system
- Prefabs for asteroids, bullets, and player
---
## 🚀 How to Run

1. Clone or download the project
2. Open Unity Hub
3. Add the project folder
4. Open the main scene (e.g. `GameScene`)
5. Press Play in Unity Editor
